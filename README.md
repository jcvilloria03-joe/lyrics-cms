# Song Lyrics CMS (PHP)

A simple content management system built using native PHP and MariaDB.

## Features
- Create, edit, delete song lyrics
- List all songs with artist and creation date
- Secure database access using PDO
- Input validation and output escaping

## Requirements
- PHP 8.1.6
- 10.4.24-MariaDB - mariadb.org binary distribution
- Apache (XAMPP)

## Setup
1. Clone repository
2. Create database `lyrics_cms`
3. Import table schema
4. Update `config.php` to the correct username and password
5. Access via browser
